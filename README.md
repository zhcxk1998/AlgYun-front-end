# ALGYun v1.0

## 简介
ALGYun 是后端程序基于python语言开发，使用django框架，web前端使用react开发，小程序端使用Taro框架开发的集学生二手交易，学生互帮互助信息，学生兼职信息发布等于一体的多功能平台

## 技术选型
环境：ubuntu 16.04 + nginx 1.10.3 + uwsgi 2.0.17.1

后端：python3.6（服务器运行环境） + django 2.1.2

web前端：React

小程序端：Taro
